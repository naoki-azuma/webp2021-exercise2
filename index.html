html要素の紹介

"ul" 要素

ul要素は順序なしリストを作成する要素です。 <ul>はいくつかの<li>を子要素に持ちます。 <li>要素にリストに記述したい内容を書きます。 また<ul>要素ではデフォルトで先頭にブレットと呼ばれる・が表示されます。 このブレットはcssのlist-style-typeプロパティを使うことで変更することができます。

example

<ul>
   <li>milk</li>
   <li>eggs</li>
   <li>bread</li>
   <li>humms</li>
</ul>

result

  • milk
  • eggs
  • bread
  • humms

"img" 要素

img要素は画像を表示する要素です。 src属性に表示したい画像のパスを書きます。 alt属性にはwebサイトがうまく画像を表示できなかった際に表示するテキストを書きます。 また、widthやheightなどの属性を指定すると画像を任意のサイズに指定することができます。

example

<img src="images/sweets_maritozzo_ichigo.png" alt="いちご入りのマリトッツォ">

result

"a" 要素

a要素はリンクを作成する要素です。 href属性にリンク先にしたいパスを書きます。 リンクにしたいものをa要素に囲んで書きます。 今回は"Google"というテキストをリンクにしています。 テキスト以外にもimg要素をリンクにすることもできます。

example

<a href="https://google.co.jp">Google</a>

result

Google
